# Crudify

Crudify is a full-stack MERN application developed to perform complete CRUD (Create, Read, Update, Delete) operations through a modern and responsive web interface. The project demonstrates full-stack development concepts including frontend integration, backend API development, database management, and state handling.

The application allows users to manage data efficiently with seamless interaction between the frontend and backend.

---

## Features

- Create new records
- View stored data
- Update existing records
- Delete records
- Responsive user interface
- RESTful API integration
- Real-time data handling
- Organized frontend and backend structure

---

## Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/crudify.git
```

---

### Navigate to the project folder

```bash
cd crudify
```

---

## Backend Setup

```bash
cd backend
npm install
npm start
```

The backend server will run on:

```bash
http://localhost:5000
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm start
```

The frontend application will run on:

```bash
http://localhost:3000
```

---

## Environment Variables

Create a `.env` file inside the backend directory and add:

```env
MONGO_URI=your_mongodb_connection_string
PORT=5000
```
